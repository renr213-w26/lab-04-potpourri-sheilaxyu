Question 1

  1. Add each of strings below to the code chunk provided in your document, render the document, and determine if the string is a proper code chunk label. If not, explain why and describe how you could fix it so the document renders.

  2. Chunk label 1:

#| label: a-label 
#| with-a-line-break

This is not a proper chunk label because it is split across two lines. Chunk labels must be defined on a single line, so the code can be fixed by combining the label into one line, such as:

#| label: a-label-with-a-line-break
  • Chunk label 2:
#| label: areaaaaaaaaaaaaaaaaaaaaaallllllllllllllllllyyyyylooooooooooooooooooonglabel

This chunk label works.

  • Chunk label 3:
#| label: label with spaces

This chunk label will not work as Quarto treats labels like IDs, so spaces break things. To fix this, we could use underscore or dashes to connect different words, such as:

#| label: label_with_spaces
  • Chunk label 4:
#| label: label-with-dashes

This chunk label works.

Tip
